    • AnalyzLog

    Log file analyzer, with full report.

    Github page

    What it does

    • read and validate a .log file,
    • analyze log levels: INFO, WARNING, ERROR,
    • detect invalid or corrupted lines,
    • analyze the overall status of the service,
    • detect recurring errors and warnings,
    • automatically generate a structured text report.

    This project allowed me to further develop my reasoning and logic in order to transform raw data into organized and understandable information. Here, the focus is placed on the project structure, business logic, and clear documentation. These are important concepts in a professional environment which, I hope, reflect my commitment and motivation.

    This project does not rely on external Python modules: it simply takes a file as input and produces a complete report as output. The goal was to organize raw data according to a defined schema and to build a modular structure so that the code remains independent and maintainable. AnalyzLog is a tool that can of course be improved, with better structural compatibility, more comprehensive testing, and more precise feedback to be implemented.
